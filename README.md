# Ubuntu

### Add a Figlet
```bash
sudo apt install figlet
sudo apt install ruby-full -y
sudo apt install lolcat
sudo apt install vim
```
Add to ~/.bashrc
```bash
figlet SHUBH | lolcat --seed 105
```
